# 상담 기록 관리 웹앱

학원 대면 상담 내용을 웹에서 기록하고, 엑셀로 업로드/다운로드하며, 기본 통계를 확인하는 Next.js 앱입니다.

## 포함 기능

- 상담 기록 직접 입력/저장
- 엑셀 템플릿 다운로드
- 엑셀 업로드(`.xlsx`, `.xls`) 후 행 단위 검증
- 현재 데이터 엑셀 다운로드
- 요약 통계(총 건수, 월 건수, 총/평균 상담시간)
- 분포 분석(성별, 연령, 상황)
- 상담 목록 확인/개별 삭제/전체 삭제

## 데이터 항목

- 날짜
- 이름
- 학원아이디
- 연락처
- 방문경로(홈페이지/인스타그램/블로그/유튜브/기타)
- 성별(남/여)
- 연령(20~24/25~29/30~34/35~39/40~45)
- 주소
- 상황(대학/졸업/직장/기타)
- 대학교
- 전공
- 학년
- 영어
- 궁금한점
- 준비기간
- 상담시간(분)
- 비고

## 실행 방법

### macOS

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.

### Windows

1. `Node.js LTS` 설치
2. 저장소 다운로드 또는 `git clone`
3. 프로젝트 폴더에서 아래 중 하나 실행

```bat
run_windows.bat
```

또는 직접 실행:

```bat
npm install
npm run dev
```

서버가 꼬였을 때는 아래 파일로 재실행할 수 있습니다.

```bat
restart.bat
```

상세한 윈도우 설치 절차는 [WINDOWS_SETUP.md](/Users/min/Desktop/consulting/WINDOWS_SETUP.md) 참고.

## 참고

- 저장소는 브라우저 `localStorage`를 사용합니다.
- 맥에서 저장한 데이터는 윈도우 브라우저로 자동 이동되지 않습니다.
- 서버 DB(Supabase/PostgreSQL) 연결 버전으로 확장하려면 API Route와 DB 스키마를 추가하면 됩니다.
